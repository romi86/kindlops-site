# Kindl Ops Website

AI-powered boutique consultancy landing page.

## Local Development
```bash
npx serve public
```
Then open http://localhost:3000

## Deploy to Vercel
```bash
npm i -g vercel
vercel
```

## Connect Domain
After deploying, go to Vercel → Settings → Domains → Add kindlops.com

## File Structure
- `public/index.html` — The full landing page (single file, no dependencies)
